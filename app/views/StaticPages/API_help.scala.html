Disponibilização de API

A API foi desenvolvida para facilitar o acesso aos dados disponíveis em várias tabelas, como cursos, disciplinas, grupos, projetos e utilizadores.

Como Utilizar

A API disponibiliza os dados de busca no formato JSON. Confira as URL base de acesso a API:

  • http://localhost:9000/API/V1/[entidades].json
  • http://localhost:9000/API/V1/[entidades]/[entidadeID].json

O parâmetro [entidades] aceita cinco possíveis valores:

  • courses
  • disciplines
  • groups
  • projects
  • users

O parametro [entidadeID] está relacionado com o id da entidade que você deseja obter.

Exemplo: http://localhost:9000/API/V1/disciplines.json
